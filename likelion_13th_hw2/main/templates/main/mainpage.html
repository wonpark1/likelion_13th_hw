views.py 작성 & HTML 페이지 연결

views.py에서는 render 함수를 이용해 템플릿을 호출합니다.

from django.shortcuts import render def mainpage(request): # 함수 선언 return render(request, 'main/mainpage.html') def secondpage(request): # Django가 secondpage.html 파일을 찾아서 연결 return render(request, 'main/secondpage.html')

템플릿 언어 (Django Template Language)

  1. 템플릿 필터 — 앞서 배운 템플릿 변수를 특정 서식에 따라 변환하는 것 (약 30여 가지 내장 필터가 제공됩니다)
